# Server-identifier
Tera proxy mod , simply it shows the name of other people server (EU).

![alt text](https://i.imgur.com/F4bKSem.png)

 ######### Command ########
 
  *  <strong>SI</strong> or <strong>Serveridentifier</strong> to Enable or Disable , by default it's Enabled
